in permissions tab 

first option will be to select roles.

If SUPERADMIN is selected (radio button saying "All permissions" then all permissions will be granted and no other options will be available.
If Not SUPERADMIN ("Specific permissions" radio buttong then user will be able to select checkboxes for 

standard roles (creator, viewer, approver, security admin)

If check box option "advanced permissions" is selected then next section is revealed
